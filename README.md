## Introduction

基于QGIS平台以插件方式开发的的高分辨率遥感影像滑坡自动识别检测系统，目前已实现多尺度分割功能，以及系统架构设计。

## Usage

1. Star 本项目
2. Clone 到本地
3. 安装QGIS软件
3. 用VS2005导入工程
4. 编译执行
5. 把生成的动态链接库拷贝至QGIS插件文件夹中
6. 打开QGIS平台，在插件菜单栏中选中我们编写的插件“高分辨率遥感影像滑坡提取”，加载进来即可!
7. Enjoy it！
